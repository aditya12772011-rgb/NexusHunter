# 🛡️ Nexus - Advanced Security Threat Analyzer

Nexus is an interactive, menu-driven command-line security auditing tool written in Python. It helps developers, system administrators, and security researchers profile web infrastructure, detect firewalls, audit security headers, scan open ports, and extract threat intelligence from public WHOIS records.

---

## 🚀 Features

- **🔴 Aggressive Red UI Banner:** Sleek and professional cyber-security themed dashboard.
- **🌐 Core Infrastructure Scan:** Resolves IP addresses, grabs HTTP server banners, and checks SSL certificate expiry.
- **🧱 Firewall/WAF Detection:** Identifies popular protection layers like Cloudflare, Akamai, AWS CloudFront, and Sucuri.
- **🔒 Security Headers Audit:** Inspects critical defensive headers (`HSTS`, `CSP`, `X-Frame-Options`, `X-Content-Type-Options`).
- **🔌 Network Port Scanner:** Quickly scans popular service ports (`21`, `22`, `23`, `25`, `80`, `443`, `8080`) to find open attack surfaces.
- **🤖 Robots.txt Analyzer:** Extracts disallowed paths and potentially sensitive hidden directories.
- **📂 Interactive Menu:** Choose to run specific modules or execute a full suite scan simultaneously.
